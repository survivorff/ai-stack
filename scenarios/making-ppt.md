# 📊 做 PPT / 演示

> 职场人最高频的"低创作"任务之一。**AI 让"做 PPT"从半天变成 30 分钟**。

---

## 你为什么读到这里?(痛点)

- **手做半天到一天**：从空白页开始排版 + 找图 + 调格式
- **模板不专业**：自己做的"看着就不对"但说不出哪里不对
- **改老板意见反复**：改 3 轮才定稿，每轮改排版又崩
- **配图找不到合适的**：免费图库质量差，付费图库贵
- **排版审美不够**：不是设计师，但要做出"看着专业"的 PPT

---

## 推荐方案

### 🥇 方案 A：AI PPT 一键生成（主推）

**适合**：工作汇报 / 述职 / 培训 / 提案 / 答辩
**时间**：30 分钟
**工具**：[Gamma](../tools/ppt/gamma.md)（海外 / 现代风）/ [美图 AI PPT](../tools/ppt/meitu-ai-ppt.md)（国内 / 职场风）

**步骤**：
1. 用 Claude / DeepSeek 写一份**简单大纲**（每章 1-3 句话核心）
2. 粘到 Gamma / 美图 AI PPT
3. 选模板（根据场景挑）
4. 一键生成 8-15 页 PPT
5. 微调内容 / 配图 / 排版
6. 导出 PPT / PDF / 在线分享

### 🥈 方案 B：Claude / DeepSeek 写内容 + 手动排版

**适合**：已有 PowerPoint / Keynote 模板，只缺内容
**时间**：1-2 小时

**步骤**：
1. 让 AI 写每页的"标题 + 3-5 个要点 + speaker notes"
2. 你粘到已有模板里
3. 手动调排版

### 🥉 方案 C：从已有文档转 PPT

**适合**：已经有 PRD / 周报 / 报告，要变成 PPT
**工具**：Gamma "Transform" / 美图 AI PPT "文档转 PPT"

**步骤**：
1. 复制文档进工具
2. AI 自动拆 slide + 配图 + 排版
3. 你 review 改细节

## 关键 Prompt 模板

### Prompt 1：写 PPT 大纲（给 AI PPT 工具用）

```
我要做一份 PPT：
- 主题：[具体主题]
- 场合：[述职 / 汇报 / 培训 / 提案 / 答辩]
- 听众：[老板 / 客户 / 同事 / 学生]
- 时长：[5 分钟 / 15 分钟 / 30 分钟]
- 风格：[商务正式 / 轻松 / 学术 / 创意]

帮我写一份 PPT 大纲：
- 每页：标题 + 3-5 个要点（每个要点 < 15 字）
- 总共 [8-15] 页
- 结构：开头(钩子) → 正文(3-5 个核心章节) → 结尾(总结 + CTA)
- 每页建议配什么图（一句话描述）

要求：
- 每页信息密度适中（不要塞太多字）
- 关键数据 / 对比 / 图表标出来
- 不要"首先 / 其次 / 综上"塑料结构
```

### Prompt 2：从文档提炼 PPT 要点

```
这是我的文档（周报 / PRD / 报告）：[贴]

帮我提炼成 PPT 要点：
- 每页一个核心信息
- 每页 3-5 个 bullet（每个 < 15 字）
- 标出"必须用图表展示"的数据
- 标出"可以删掉不影响理解"的内容
- 总共控制在 [10-15] 页

输出格式：
Page 1: [标题]
- 要点 1
- 要点 2
- 配图建议：[...]

Page 2: ...
```

### Prompt 3：写 Speaker Notes

```
这是我的 PPT 大纲：[贴每页标题 + 要点]

帮我写每页的 speaker notes（我演讲时说的话）：
- 每页 3-5 句话
- 口语化（不是读 PPT 上的字）
- 包含"过渡句"（从上一页到这一页怎么衔接）
- 标出"这里停顿 / 这里强调 / 这里看观众"

时长目标：整套 [15] 分钟。
```

### Prompt 4：改老板意见后快速调整

```
老板看了我的 PPT 给了以下反馈：
1. [反馈 1]
2. [反馈 2]
3. [反馈 3]

我的 PPT 大纲是：[贴]

帮我：
1. 逐条分析老板真正想要什么（有时候反馈 ≠ 真实需求）
2. 给出修改方案（每条反馈对应改哪几页）
3. 改完后的新大纲

不要全部推翻重来，最小改动满足反馈。
```

## 真实使用 tips

### 1. 先写大纲再生成（不要直接"帮我做 PPT"）

❌ 直接让 AI PPT 工具"帮我做一份关于 X 的 PPT"
✅ 先用 Claude / DeepSeek 写好大纲 → 再粘到 Gamma / 美图

**为什么**：AI PPT 工具擅长"视觉"，不擅长"内容策略"。
**内容策略用 Claude / DeepSeek，视觉用 Gamma / 美图**。

### 2. 每页只说一件事

PPT 不是文档。**每页一个核心信息**：
- ❌ 一页塞 8 个 bullet + 3 段话
- ✅ 一页 1 个标题 + 3-5 个短 bullet + 1 张图

**听众看 PPT 的时间 = 3-5 秒/页**。超过就走神。

### 3. 数据用图表，不用文字

```
❌ "我们 Q1 营收增长 23%，用户增长 45%，留存提升 12%"
✅ 一张柱状图 / 折线图，3 秒看懂趋势
```

让 AI 帮你选图表类型：
```
我有这些数据：[贴]
帮我选最合适的图表类型 + 画 mermaid / 描述给设计师。
```

### 4. 配图用 AI 生成（不要找网图）

- [即梦](../tools/image-gen/jimeng.md)（国内免费）
- [Midjourney](../tools/image-gen/midjourney.md)（艺术感）
- Gamma 自带 AI 配图

**AI 配图 vs 网图**：
- 不撞图（每次生成都是新的）
- 风格可控（和 PPT 模板匹配）
- 不用担心版权

### 5. 一键换风格（同内容不同场合）

Gamma / 美图 AI PPT 顶部"换风格"按钮：
- 同内容 → 商务蓝
- 同内容 → 创意橙
- 同内容 → 学术白

**一份内容，3 个场合，3 分钟切换**。

### 6. 国内 vs 海外选哪个

```
国内职场 / 政企 / 教育 / 答辩：  美图 AI PPT（模板对路）
海外项目 / 创业 BP / 现代风：    Gamma
极致定制 / 发布会：              手作 PowerPoint / Keynote + 设计师
```

## 真实案例

我做 content-os 项目的内部分享 PPT：
- Claude 写大纲（10 分钟）
- 粘到 Gamma → 生成 12 页（5 分钟）
- 微调 3 页内容 + 换 2 张配图（15 分钟）
- **总 30 分钟**
- 手做估计 3 小时

---

## 这套方案的边界

⚠️ 什么时候不适用：

- **发布会 / 资本路演级**：仍要设计师 + 手作（AI 做不到"惊艳"）
- **极复杂动画 / 交互**：PowerPoint 动画 AI 工具简化了
- **极度个性化品牌**：品牌 VI 严格的公司，AI 模板可能不合规
- **1-2 页的简单 slide**：不需要工具，直接 PowerPoint 手做
