# 🇨🇳 国内可用性速查

> 这是 ai-stack 的核心差异化字段之一。
> 所有工具笔记都标注 china_availability,这里汇总。

---

## 标注规则

| 标记 | 含义 |
|------|------|
| 🟢 直接可用 | 国内网络 + 中国信用卡可付。零配置。 |
| 🟡 需要自配 | 需要代理 / 海外卡 / API key。**不是不能用,是要折腾**。 |
| 🔴 不可用 | 国内网络封锁严重,或地区限制不卖中国 |
