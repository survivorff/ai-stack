## 一人公司的"AI 杠杆点"

最值得让 AI 替代你的 4 件事:

1. **写文案 / 写长文** → Claude / DeepSeek 替代你 50% 写作时间
2. **调研 / 信息整理** → Gemini Deep Research / Kimi 探索版替代 80% 时间
3. **重复邮件 / 客服 / 周报** → Custom GPT / 飞书 AI 替代 90% 时间
4. **MVP 写代码 / 数据分析** → Cursor / Claude Code / Aider 替代 70% 时间

**剩下的 4 件事 AI 暂时替代不了**,这是你的核心:
- 战略决策(产品方向 / 商业模式)
- 销售转化(关键客户洽谈)
- 团队 / 招聘(人际判断)
- 创意 / 真正原创性的内容
