## 不同档位的程序员推荐组合

### 学生 / 入门(月预算 ¥0)

```
通义灵码 / 文心快码           — 主 IDE 编程辅助
DeepSeek                    — 通用问答
豆包                        — 即时问题
```

### 个人开发者 / 月薪 1-3 万(月预算 ¥150-200)

```
Cursor Pro $20/mo            — 主 IDE
Kiro 免费版                  — Spec 驱动
DeepSeek / Kimi             — 备用问答
```

### 重度 + 多项目并行(月预算 ¥500-1000)

```
Cursor Pro $20/mo
Claude Code Pro $17/mo + API — 长任务
Kiro 付费版                  — 团队协作
ChatGPT Plus $20/mo         — 多模型对比
```
