## 当前 9 大角色

| 角色 | 状态 | 简述 |
|------|:---:|------|
| 💻 [程序员 / 工程师](./developer/) | ✅ 已上线 | 写代码、debug、读老项目、写文档 |
| 📋 [产品经理](./product-manager/) | 🟡 起步中 | 写 PRD、用户调研、竞品分析 |
| 🎨 [UI/UX 设计师](./designer/) | 🟡 起步中 | 概念稿、配图、Logo、原型 |
| 📣 [运营 / 营销](./marketer/) | 🟡 起步中 | 内容、文案、SEO、数据 |
| 🎬 [自媒体 / 创作者](./creator/) | 🟡 起步中 | 脚本、视频、配音、封面 |
| 🚀 [创业者 / 一人公司](./founder/) | 🟡 起步中 | BP、市场分析、法务 |
| 💼 [职场白领 / 办公](./office-worker/) | 🟡 起步中 | PPT、会议纪要、邮件 |
| 🎓 [学生 / 学习者](./student/) | 🟡 起步中 | 论文、读书、刷题 |
| 👨‍🏫 [教师 / 教育者](./educator/) | 🟡 起步中 | 教案、批改、课件 |

> ✅ = 完整角色页 + ≥5 场景  
> 🟡 = 仅有目录骨架,需逐步填充(月度更新中)
