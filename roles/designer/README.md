## 不同档位的设计师推荐组合

### 入门 / 学生(月预算 ¥0)

```
即梦 / 通义万相 / 文心一格(国产 + 免费)
+ 秘塔 / Kimi(调研)
+ Figma 免费版 + AI 插件
```

### 个人 / 自由设计师(月预算 $50-100)

```
Midjourney Standard $30      — 艺术 / 风格化
+ Figma Pro $15               — 主设计
+ Adobe Creative Cloud(部分) — Illustrator / Photoshop 仍要
+ Claude Pro $20              — 写 design 文档
```

### 设计公司 / 团队(月预算 $200+)

```
Midjourney Pro $60(stealth)
+ Figma Organization
+ Adobe CC 全家桶
+ FLUX Pro(API,国内大批量)
```
