## 重要警示

⚠️ **AI 写论文 + 降 AI 检测**这个赛道我们**不收录**。理由:
- 学术诚信问题
- 越来越多的检测工具会反向识别"降重"内容
- 这是教育系统的红线

我们做的是"**AI 辅助你真正学懂**",不是"绕开学术诚信"。

## 学生的"零成本 AI 工作流"

**月费 ¥0,学习能力质变的工具组合**:

```
通义灵码 + 沉浸式翻译(必装,免费)
+ DeepSeek + Kimi + 豆包(三大主力,免费)
+ 秘塔(调研,免费)
+ Obsidian(本地笔记,免费)

```

**这套组合 vs 海外学生用的 ChatGPT Plus + Notion + Grammarly $50+/月** —— 90% 场景对齐,而且零门槛。

## 实战场景 tips

### 啃论文

```
1. 沉浸式翻译看英文 paper 双语
2. 重点章节用 Kimi 上传 PDF 精读
3. 学术问题用秘塔学术模式追问
4. 笔记沉淀到 Obsidian
```

### 学一个新领域

```
1. 用 DeepSeek / Claude 让它给"骨架 + 学习路径"
2. 收集 10 篇相关资料 → Kimi / NotebookLM 上传
3. 让 AI 出综述 + 关键概念
4. 边学边在 Obsidian 写笔记
```

### 写毕业论文

```
1. 用 Kimi 探索版 + 秘塔学术做调研(找文献)
2. 用 Claude / DeepSeek 帮整理大纲
3. 自己写主体内容(学术诚信底线)
4. 用 DeepL Write / Grammarly 改英文
5. 用美图 AI PPT 做答辩 PPT
```

### 学编程

```
1. 装 Cursor Free + 通义灵码
2. 跟 AI 一边写一边问"为什么这么写"
3. 不要让 AI 写完整作业(自己学不到)
4. 项目卡住时用 Claude / DeepSeek 解释概念
```
