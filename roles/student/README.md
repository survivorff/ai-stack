## 重要警示

⚠️ **AI 写论文 + 降 AI 检测**这个赛道我们**不收录**。理由:
- 学术诚信问题
- 越来越多的检测工具会反向识别"降重"内容
- 这是教育系统的红线

我们做的是"**AI 辅助你真正学懂**",不是"绕开学术诚信"。
