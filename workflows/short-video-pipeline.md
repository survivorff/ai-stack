# 🎬 短视频脚本 → 制作 → 发布完整流水线

> 自媒体 / 营销 / 创业者:**从灵感到发布的全 AI 流水线,2 小时出片**。

---

## 这个工作流解决什么

详见 [→ 场景:短视频](../scenarios/short-video.md)。

简化重述:你想:
- 做**抖音 / 小红书 / 视频号 / B 站 / YouTube Shorts** 的短视频
- 没专业摄影 / 演员 / 录音棚
- 不想自己出镜 / 录音
- 1 个人 / 半天 / 出 1-3 条片
- **质量在"看着不像 AI 糊弄" 这一档**

---

| 阶段 | 工具 | 必备 / 可选 |
|------|------|:----------:|

## 完整流程总览

```
Step 1  ─ 选题 + 脚本 ────► AI 出脚本 + 你定调
Step 2  ─ 分镜 ──────────► 把脚本拆成"每 5-10 秒一镜"
Step 3  ─ 画面 ──────────► AI 视频生成(Kling / Hailuo)
Step 4  ─ 配音 ──────────► Speech-02 克隆你声音 + 配
Step 5  ─ BGM ───────────► Suno / Music-02
Step 6  ─ 剪辑 + 字幕 ────► 剪映合成
Step 7  ─ 封面 ──────────► AI 出图 + 文字嵌入
Step 8  ─ 发布 + 数据复盘 ─► 多平台错峰
```

**总耗时**:1 条精品 30-60 秒短视频 ≈ 2-4 小时(传统团队估 1-2 天)。

## Step 1:选题 + 脚本

**工具**:Claude / DeepSeek

### 1a. 选题(挑能爆的)

```
我做 [自我描述,如:链上工程师 / UI 设计师 / 大学生]。
我的人设是 [一句话定位]。
我做 [抖音 / 小红书 / B 站] 短视频,目标受众 [具体]。

帮我出 10 个本周可拍的选题:
- 每个选题:标题 + 钩子(开头 5 秒说什么) + 大致内容 + 预计时长
- 评估每个的"爆款潜力"(1-10):流量趋势 + 受众匹配 + 我的差异化
- 平台适配:抖音偏 [X] / 小红书偏 [Y] / 哪些适合哪个

最后推荐 1-2 个,说理由。
```

### 1b. 脚本(钩子 + 干货 + 结尾)

确定选题后:

```
帮我写 [选题] 的短视频脚本:

平台:[抖音 / 小红书 / B 站 / YouTube Shorts]
时长:30-60 秒
目标:[涨粉 / 引流 / 卖货 / 传达观点]

结构:
- **0-5 秒钩子**(让人不滑走的一句话)
- **5-50 秒干货**(3-4 个核心观点 / 步骤,每个 8-15 秒)
- **50-60 秒 CTA**(关注 / 评论 / 主页)

要求:
- 口语化,不要书面语
- 每句话 < 15 字(短视频节奏)
- 关键词重复 2-3 次(算法权重)
- 数据 / 反差 / 故事(让人记得住)
- 不要"AI 味"塑料话术

输出格式:
- 时间码 / 旁白 / 画面建议 / BGM 节奏建议
```

⚠️ **关键判断**:**钩子不行,后面再好都没用**。
钩子 prompt 单独迭代:

```
我的 0-5 秒钩子是:[XX]。
帮我做 5 个变种,各自的"刺点":
- 反差型("我以为 X,结果 Y")
- 数字型("3 件事改变了 Z")
- 痛点型("如果你也 [痛点]...")
- 反常识型("99% 的人都搞错了 X")
- 故事型("3 年前我犯了一个错...")
```

→ 选最对你"内容内核"的钩子。

---

## Step 2:分镜(脚本 → 每 5-10 秒一镜)

**工具**:Claude / DeepSeek

```
基于 [Step 1 脚本],拆成分镜表:

| 时间 | 旁白 | 画面 | 镜头 | BGM 节奏 |
|------|------|------|------|---------|
| 0-3s | "..." | [画面描述] | 推 / 拉 / 摇 / 固定 | 节奏 |
| 3-7s | ... | ... | ... | ... |

要求:
- 每个画面"AI 视频生成 prompt 友好"(具体 / 可视觉化 / 无歧义)
- 镜头变化(不要 60 秒固定一个镜头 — 太单调)
- 关键画面要有"信息增量"(不只是配旁白)

输出:可以直接拿去 Kling / Hailuo prompt 的画面描述列表。
```

⚠️ **每个画面都要画面描述**(给 AI 视频生成的)。
**别让 AI 生成时自己想 — 你定画面**。

---

## Step 3:画面(AI 视频生成)

**工具**:Kling 3(国产 default)/ Hailuo 02(物理感强)/ Sora 2(海外顶级)

### 3a. 写 prompt(给视频 AI)

每个分镜画面单独 prompt:

```
镜头:[运镜:从近景到远景缓慢推开]
主体:[具体描述,如:女子手拿茶杯,蒸汽缓缓升起]
环境:[傍晚咖啡馆,暖光,远处人模糊]
风格:[日系治愈,浅景深,温暖色调]
时长:[5 秒 / 10 秒]
比例:9:16(短视频竖屏)
```

⚠️ **"运镜 + 物理感"** 是 prompt 关键。"美丽女子喝茶"出来的视频质量低 5 倍。

### 3b. 多生成几次挑

每个分镜:
- 生成 2-3 次(免费档够)
- 选最稳定的那条
- **不达标 → 改 prompt 再生成**

### 3c. 配套静态图(开头帧 / 封面)

如果某些镜头要"先有静态图,然后让它动":
- [Midjourney](../tools/image-gen/midjourney.md) / [Nano Banana](../tools/image-gen/nano-banana.md) / [即梦](../tools/image-gen/jimeng.md) 出图
- Kling "图生视频"功能,让静态图动起来
- **比"纯文生视频"质量稳得多**

---

## Step 4:配音(让 AI 用你的声音)

**工具**:Speech-02(国产中文)/ ElevenLabs(英文)

### 4a. 克隆你自己的声音

第一次做:
- 录 30-60 秒清晰说话(普通话标准 + 单一情绪)
- 上传到 Speech-02 / ElevenLabs
- 立即生成"你的克隆声音"

之后每个视频:**配音用你的声音**(听起来还是你)。

### 4b. 配音生成

```
旁白文本:[Step 1 / Step 2 的旁白]
要求:
- 语速:中速(短视频)
- 情感:激动 / 平静 / 神秘 / 严肃(看场景)
- 停顿:符号到位
- 重音:[关键词上重音]
```

→ 输出 mp3。
→ 如果有"语气问题"重新生成。

⚠️ **TTS 文本要加标点和"!"".."等节奏标记** —— 出来的语气更自然。

### 4c. 不想露声音?用预设音色

Speech-02 / ElevenLabs 都有几十个预设音色:
- 选一个匹配你视频调性的(治愈 / 严肃 / 活泼)
- 不必克隆

短视频不一定要"你的声音",**调性匹配比身份重要**。

---

## Step 5:BGM(背景音乐)

**工具**:Suno(海外)/ Music-02(国产 / 海螺)

### 5a. 让 AI 写 BGM

```
描述给 Suno:
"30 秒 BGM,治愈风格,木吉他 + 钢琴,80 BPM,
情绪从平静到温暖,无人声,适合自媒体短视频背景。"
```

→ 出 1-2 首 → 选合适的。

### 5b. 或用免费曲库

- 抖音 / 剪映自带免费 BGM 库(基本够日常)
- YouTube Audio Library(海外免费)
- Pixabay Music(免费 + 商用)

**对入门创作者:免费曲库够 80% 场景**。
**真做专业内容:Suno + 商用 license**。

⚠️ **BGM 音量要低**(背景,不抢主音)。剪映 / Final Cut 调到 -20dB 左右。

---

## Step 6:剪辑 + 字幕

**工具**:剪映(国内 default,免费)/ Final Cut(macOS)/ DaVinci(免费高级)

### 6a. 剪映工作流

```
1. 新建项目(竖屏 9:16)
2. 导入:Step 3 视频片段 + Step 4 配音 + Step 5 BGM
3. 视频片段按分镜表排序
4. 用"自动节拍"对齐 BGM 和画面切换
5. 加 AI 字幕(剪映自带,识别准确度高)
6. 字幕样式:大 / 醒目 / 关键词加色
7. 加转场(简单 — 短视频不要花哨转场)
8. 加封面(下面 Step 7)
9. 导出(1080P / 60fps,如果原素材支持)
```

### 6b. 字幕是关键

**60-80% 抖音 / 小红书用户没声音看视频**。字幕 = 第二条信息线。

字幕处理:
- 剪映 AI 字幕识别 + 你 review 改错字
- 关键词加颜色 / 加大
- 不要把整个旁白都做字幕(太密),**只字幕重点**

### 6c. 节奏剪辑

短视频"快节奏"很重要:
- 每 2-4 秒切一个画面
- 关键句配视觉冲击(放大 / 闪烁 / zoom in)
- 静止超过 3 秒 = 用户开始划走

---

## Step 7:封面(决定点击率)

**工具**:Midjourney / Nano Banana / 即梦 + Photoshop / 剪映 / Canva

### 7a. 封面图生成

```
prompt:
[一句话视觉冲击]
+ 高对比度(关键)
+ 9:16 竖屏
+ 大块留白给文字
+ 风格:[符合你账号定位]
```

### 7b. 文字嵌入

封面文字要:
- **3-7 个字**(超过 8 字读者不看)
- 反差 / 数字 / 钩子("只看这 3 件")
- 字号大、加粗、对比强
- 用剪映 / Canva 加文字

⚠️ **封面文字 ≠ 标题**。封面文字更短更冲击。

### 7c. A/B 测试封面

抖音 / 小红书允许换封面 + 看数据。
做 3-5 个候选 → 发布后看哪个 CTR 高 → 切到最好的那个。

---

## Step 8:发布 + 数据复盘

### 8a. 多平台错峰发

```
Day 1 早:抖音(国内黄金时段)
Day 1 中:小红书(白天高峰)
Day 1 晚:视频号(微信生态)
Day 2:B 站 / YouTube Shorts(海外)
```

### 8b. 标题 + 文案

每个平台单独写:
- **抖音**:钩子型标题 + #标签 +引导评论
- **小红书**:反差 / 教程 / 痛点型 + 大量 #标签
- **视频号**:微信熟人风,稍正式
- **YouTube Shorts**:英文 + 钩子 + 关键词

### 8c. 数据复盘(7 天后)

```
平台 / 播放量 / 点赞率 / 评论 / 涨粉:
抖音    XXXX   X%      XX     +XX
小红书  XXXX   X%      XX     +XX
视频号  XXXX   X%      XX     +XX

发现:
- 哪个平台数据最好?为什么?
- 哪个时段播放高?
- 评论里读者最关心什么?
- 下期可以延伸什么?
```

让 AI 帮你分析评论:

```
这是这条视频的评论(50-200 条):[贴]

帮我:
1. 分类(正面 / 负面 / 中性 / 提问)
2. 提炼"最高频问题"(下期内容选题)
3. 找"金句评论"(可以做 cover)
4. 找"潜在合作 / 交易意向"
5. 给我"应该回复哪些评论"清单
```

## 这个工作流的边界

⚠️ **什么时候这套不够**:

1. **真人出镜 / 真实场景**
   - AI 视频生成做不了"完全真实的某地某时"
   - 旅游 vlog / 真人脱口秀 → 仍要拍
2. **极复杂特效 / 长视频**
   - 5 分钟 + 复杂剧情 → AI 视频还撑不住
   - 用 AI 出片段 + After Effects 做特效
3. **品牌广告级 / 商业级**
   - 大品牌 commercial → 仍要专业拍摄 + 设计师
   - AI 出"方案 / 概念 / sketch",最终人来执行
4. **极强人设依赖**
   - 美妆 / 健身 / 时尚博主必须真人出镜
   - AI 不可代替"真实身体感"

## 进阶变种

### 变种 A:**纯 AI / 不出镜 / 不录音**

- 全 AI 生成画面 + 克隆配音 + AI BGM
- 适合"我只想做内容,不想暴露脸 / 声音"
- 我自己的"链上工程笔记"系列就是这套

### 变种 B:**真人 + AI 增强**

- 自己手机拍真人讲解
- 剪映 AI 加"画面增强 / 字幕 / 转场 / 配乐"
- 适合"已有真人内容,想用 AI 提质"

### 变种 C:**视频自动化**(高频创作者)

- 写好 spec(每周话题 / 风格 / 结构)
- AI 自动生成脚本 + 视频 + 配音 + BGM
- 你只 review 关键决策(选题 / 钩子)
- 出片量翻 3-5 倍

工具:[content-os](https://github.com/survivorff/content-os) 之类的"创作流水线"。

## 实施前的 Checklist

发布前自查:
- [ ] 钩子(0-5s)真的能吸引你不滑走?
- [ ] 干货密度够(每 10 秒 1 个有效信息点)?
- [ ] 配音 / 字幕 / 画面 节奏对齐?
- [ ] 字幕没错字 / 没遮挡关键画面?
- [ ] BGM 音量低,不抢主音?
- [ ] 封面 7 字内 + 高对比 + 钩子?
- [ ] 标题 / 文案 每平台单独写?
- [ ] 内容合规(各平台规则各异,Web3 / 加密敏感话题别发小红书)?
