# Gamma

> AI PPT 生成的行业第一档。**输入大纲 / 文档 → 自动出整套 PPT**。
> Tier A+。**对"我要 30 分钟做完一份 PPT"的场景几乎是默认选择**。

## Tier 评级:A+

**评级理由**:
- **AI PPT 行业 default**:用户量 / 模板生态 / 体验都是海外第一
- **30 分钟一份 PPT**:粘大纲 / 文档进去,30 分钟完成对手手作 4-8h 的活
- **模板美感稳**:不会出"明显丑陋的 AI PPT"
- **Web 演示 + 导出**:可在线分享(类 Google Slides) + 导出到 PPT / PDF

**不是 S 的原因**:
- **中文模板** vs 英文略少
- **数据走云端**(协议看 Gamma 政策)
- **极致专业演示**(发布会 / 资本类)仍要手工
- **高度自定义** 需要付费档解锁

## 它最适合做什么

### ✅ 工作 PPT(周报 / 月报 / 提案 / 培训)

最适合的场景:
- 年终述职 / 项目汇报
- 新人入职培训 PPT
- 产品 / 技术分享
- 客户提案 / 投标

普通工作 PPT:**Gamma 30 分钟 vs 手工 4 小时**。

适合:
- [做 PPT](../../scenarios/making-ppt.md)
- [工作汇报](../../scenarios/work-reporting.md)

### ✅ 把已有文档转 PPT

工作流:
- 你已经有 PRD / 周报 / 报告(Word / Markdown / Notion)
- 复制文档进 Gamma → "Transform to Presentation"
- AI 自动拆 slide + 配图 + 排版

**对"已经写好文档,缺 PPT"的场景:救命**。

### ✅ 创业 BP / 个人作品集

- BP 各章节(问题 / 解决方案 / 商业模式 / 团队 / 数据)
- 个人作品集
- 产品 demo deck

模板美感稳定,**不会让你"做出来不好意思发"**。

### ✅ Web 演示 + 在线分享

Gamma 是 Web 优先:
- 一个 URL 分享给人
- 类 Google Slides 体验
- 远程会议直接展示(不用下载)

**比导出 PPT 再发邮件简洁**。

## 它**不擅长**什么

### ❌ 中文模板美感

Gamma 中文模板 vs 英文略弱。
中文场景顶级美感:[美图 AI PPT](./meitu-ai-ppt.md) / [讯飞智文](./xunfei-zhiwen.md)。

### ❌ 极致专业演示

发布会 / 投资人路演 / 学术发表:仍要手作(Keynote / PowerPoint)+ 设计师介入。
Gamma 适合"够用 + 美观",**不是顶级演示**。

### ❌ 极致动画 / 交互

PowerPoint 的动画 / 过渡 / 嵌入交互:Gamma 简化了这些。
重视动画的演示 → PowerPoint。

### ❌ 完全免费

Gamma 免费档**有 watermark + 限制**。
完全免费 → 国内 美图 AI PPT / 讯飞智文 推广期。

## 真实使用 tips

### 1. 先粘文本,再让 AI 生成

最好的工作流:
- 你写一份**简单大纲**(每章 1-3 句话核心)
- 粘到 Gamma → "Generate Presentation"
- AI 出 8-15 页 PPT
- 你 review 改细节

**比"一句话生成"质量高 3 倍**(给 AI 更多输入信号)。

### 2. 模板选对节省一半时间

Gamma 有几十个模板(business / pitch / academic / report 等):
- 选对 → 视觉自动协调
- 选错 → 改半天

**先选模板再开始**(很多人犯反向错误)。

### 3. 多语言 / 中文也行

Gamma 中英文都支持。生成时**显式选语言**:
- "Generate in 中文"
- 或者"This is a Chinese presentation"

中文输出 OK,**只是模板设计偏西方审美**。

### 4. 配色一键切

每个 PPT 顶部"Theme"按钮:
- 一键换配色 / 字体 / 风格
- 内容不变,视觉刷新

适合:
- 客户 A 提案 → 商务蓝
- 客户 B 提案 → 同内容换品牌色
- 节省"为不同客户改 PPT"时间

### 5. AI 配图(海外 Imagen)

Gamma 自带 AI 配图(走 Imagen / Nano Banana)。
- 每页自动配图
- 与 slide 主题相关
- 比"找网图免抠图"快

**专业品牌项目仍要专业图**,但日常 PPT 这一步省巨多。

### 6. 配 Claude / GPT 的工作流

```
1. Claude / GPT 写文档(完整大纲 + 内容)
2. 粘进 Gamma 生成 PPT
3. Gamma 内 review + 微调
4. 导出 PPT / PDF / 在线分享
```

**比直接"让 AI 做 PPT"质量高,因为 Gamma 擅长 PPT 视觉,Claude 擅长内容**。
