## 📊 推荐组合(2026-05)

**做 Logo:**
- **[Midjourney V7](../image-gen/midjourney.md)**(艺术 / 概念稿)
- **[Ideogram 3](../image-gen/) / [Nano Banana](../image-gen/nano-banana.md)**(文字嵌图)
- **Looka / LogoAI**(自动 Logo 生成)

**做 UI 原型:**
- **Figma + AI 插件**(主流设计师工作流)
- **v0(Vercel)**(从描述到 React UI)
- **Galileo AI** / **Magician**(Figma 内 AI 生成)
- **uizard**(草图 → UI)

**做配色 / 风格系统:**
- **Khroma**(AI 配色)
- **Coolors**(经典配色生成器,2024 加 AI)
- **Stable Diffusion + ControlNet**(自定义 / 可控)

**做营销图 / 海报:**
- **[Midjourney V7](../image-gen/midjourney.md)** / **[FLUX Pro](../image-gen/flux.md)** / **[即梦](../image-gen/jimeng.md)**

## 待详写

| 工具 | 长板 | 优先级 |
|------|------|--------|
| v0(Vercel) | 描述 → React 组件代码 | 中 |
| Figma AI | 主流设计软件原生 AI | 高(Figma 用户必看) |
| uizard | 草图 → UI(老牌) | 低 |
| Khroma | AI 配色 | 低 |
| Looka | 自动 Logo | 低 |
| Recraft | 矢量 + 高级 AI 设计 | 中 |
| Galileo AI | Figma 插件 + AI UI 生成 | 中 |

> 设计分类大量工具是 [image-gen](../image-gen/) 的"垂类化产品壳"。
> Logo / 海报 / 营销图 → 直接用 image-gen 头部工具(Midjourney / FLUX / 即梦)。
> UI 设计 / 原型 → Figma + AI 插件仍是主流。
> 因此本分类暂不深拆。
