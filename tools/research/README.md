## 🧭 怎么选

```
你要"搜索式"快速答(海外英文): → Perplexity
你要"搜索式"快速答(国内中文): → 秘塔
你要"消化已有资料 + AI 播客":   → NotebookLM
你要"30-60 分钟深度综述":       → Gemini Deep Research(见 foundation-models/gemini.md)
你要"国内 + 长文档调研":          → Kimi 探索版(见 foundation-models/kimi.md)
```

**实际工作流常常多个组合**:
- Gemini Deep Research / Kimi 出综述
- NotebookLM 做"消化 + 播客 + 长期沉淀"
- Perplexity / 秘塔 做"快速搜"
