# NotebookLM

> Google 出的"基于你上传资料的 AI 问答"。**100% 免费 + 自动生成 AI 播客版摘要**。
> Tier A+。**学习 / 研究 / 知识工作者的隐藏宝藏**。

---

## 基本信息

| 项 | 值 |
|---|---|
| **厂商** | Google(Gemini 团队) |
| **形态** | Web([notebooklm.google.com](https://notebooklm.google.com)) |
| **底层模型** | Gemini 系列 |
| **核心机制** | 上传资料 → 只基于这些资料回答 |
| **国内可用性** | 🟡 需要 Google 账号 + 海外网络 |

---

## Tier 评级:A+

**评级理由**:
- **基于资料回答(不胡编)**:这是 NotebookLM 的核心承诺 —— 它**不会"自由发挥"**,只引用你给的资料
- **自动音频播客**:**两个 AI 主播聊你上传的资料,质量惊人**(2024 年出圈)
- **完全免费**:Google 不卖,纯产品级
- **个人学习场景的天花板**

**不是 S 的原因**:
- **海外网络 + Google 账号**门槛
- **没有外部搜索**(只问你给的资料)
- **中文支持** 比英文略弱
- **Google 偶尔砍产品** 风险(虽然 NotebookLM 在 Gemini 战略上很重要)

---

## 它最适合做什么

### ✅ 学一个新领域 / 啃论文

工作流:
- 收集 10-30 篇相关 paper / 文章 / 视频
- 全部上传到 NotebookLM
- 让它做综述 / 答疑 / 出"重点笔记"
- **听一遍它生成的 AI 播客**(开车 / 跑步时)

**论文 / 综述 / 啃书的效率提 5-10 倍**。

适合:
- [论文消化](../../scenarios/paper-digesting.md)
- [学新领域](../../scenarios/learning-new-domain.md)
- [技术调研](../../scenarios/tech-research.md)

### ✅ AI 播客生成(独家杀手锏)

最被低估的功能:**Audio Overview**。
- 上传资料(任意,中英文都行)
- 点 "Generate Audio Overview"
- **2 个 AI 主播(男女)15-25 分钟讨论你的资料**
- 听感像真的播客

实测:
- 听一遍 ≈ 读一遍精华版
- 适合通勤 / 散步 / 做家务时学习
- "把任何资料变成播客" 这件事 NotebookLM 是独家

### ✅ 个人知识库

```
长期项目(论文 / 创业 / 学习)
  → 一个 NotebookLM
  → 持续添加资料(几个月几年)
  → 任何问题在这个 notebook 里问
```

类似 ChatGPT Projects / Claude Projects / Perplexity Spaces,**但完全免费 + 长上下文最好**。

### ✅ 团队 / 教学

新人入职:
- 把入职资料 / wiki / 视频 全上传一个 notebook
- 新人在这个 notebook 自助学习
- 不懂直接问 AI

老师备课:
- 上传教科书 + 论文 + 课件
- 用 notebook 自动生成讲义 / 测验题

## 它**不擅长**什么

### ❌ 联网搜索

NotebookLM **只基于你上传的资料**,**不联网搜索**。
要"AI + 实时搜索" → Perplexity / Gemini Deep Research / Kimi 探索版。

### ❌ 国内零门槛

需要 Google 账号 + 海外网络。
国内不愿折腾 → Kimi 长上下文(上传资料 + 提问)。

### ❌ 写代码 / 复杂推理

NotebookLM 是"资料问答"工具,**不是写代码工具**。
- 写代码 → Claude / DeepSeek / Cursor
- 复杂推理 → Claude Opus / GPT-5 Pro

### ❌ 极长资料(几千页书)

虽然 NotebookLM 长上下文好,但**单 notebook 资料 50+ 时质量略下降**。
分主题建多个 notebook 比一个塞超多资料更有效。

## 真实使用 tips

### 1. Audio Overview 必试

```
1. 任意上传 5-10 篇资料
2. 点 "Generate Audio Overview"
3. 等 5-10 分钟
4. 用手机听
```

**第一次听完会觉得"这就是未来学习的方式"**。

### 2. Source 类型多样

NotebookLM 接受:
- PDF / Word / TXT / Markdown
- Google Doc / Slides
- YouTube 视频(读字幕)
- 网页 URL
- 粘贴文本

**最强:YouTube 视频上传**,等于让你"看 YouTube 后立刻有 AI 总结"。

### 3. Source Guide 是被低估的

每个 source 上传后,左侧自动生成:
- 关键概念
- 主要论点
- 引用频率
- 跨 source 对比

不用你自己整理,**AI 自动建索引**。

### 4. Studio 区域生成"产物"

右侧 Studio 区域可以一键生成:
- 摘要
- 时间线
- 学习指南
- 简报文档(2-3 页 brief)
- FAQ
- **音频播客**

每个都是"基于你 source 的资料",不会编。

### 5. 配合 Gemini Deep Research

工作流:
- **Gemini Deep Research** 调研 → 出综述 + 引用源
- 把综述 + 引用源 上传到 **NotebookLM**
- NotebookLM 做"消化 + 播客 + 后续问答"

**两个工具配合,从 调研 → 学习 → 沉淀 闭环**。

### 6. 中文资料完全 OK

NotebookLM 中文支持 OK:
- 上传中文 PDF
- 中文 prompt 问答
- 音频播客也支持中文(2025 后加上)

但**英文 source 体验仍稍好**(模型训练数据更多)。

## 一段判断

NotebookLM 是 Google 在 AI 工具里**最被严重低估的产品**:
- 完全免费
- 独家"AI 播客"功能
- 是"基于资料问答"领域的实际标杆
- 学生 / 研究员 / 知识工作者**用了回不去**

**对能访问 Google 的用户,NotebookLM 几乎应该装上**。
对国内用户:**Kimi 是最近的替代**,但 NotebookLM 的"音频播客"是 Kimi 没有的独家。
