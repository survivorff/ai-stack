## Tier 评级:A+

**评级理由**:
- **中文 AI 搜索 default**:中文资料(知乎 / 公众号 / 微博 / 学术)覆盖度国产第一档
- **完全免费**:对手 Perplexity Pro 才有的"深度搜索"在秘塔免费
- **学术 / 论文场景强**:专门有"学术"模式 + 知网 / 万方等中文学术源
- **国内零门槛**:支付宝登录 + 中文母语

**不是 S 的原因**:
- **海外英文资料**vs Perplexity / Gemini Deep Research 弱

- **极致深度调研**仍弱于 Gemini Deep Research
- **海外认知度**极低

## 它最适合做什么

### ✅ 中文调研 / 信息搜索

- 查国内公司 / 行业
- 追中文新闻 / 时政
- 搜中文学术 / 论文
- 找中文专业资料

国内创作者 / 学生 / 调研人员:**秘塔是 default**。

### ✅ 学术模式

秘塔的"学术"模式:
- 优先搜知网 / 万方 / 学术网站
- 输出含 paper 引用
- 适合"做综述 / 写论文" 的中文场景

学生 / 研究员:**比 Kimi 探索版还专精**(Kimi 综合更强,秘塔学术更专)。

### ✅ 中文公众号 / 知乎深度搜

中文互联网最值钱的内容**很多在公众号 / 知乎**。
秘塔覆盖这些 vs Perplexity:**秘塔完胜**。

### ✅ 简洁体验 + 完全免费

新人:打开 → 输入问题 → 出答案。
**不需要付费 / 不需要登录(部分功能)/ 不需要折腾**。

## 它**不擅长**什么

### ❌ 海外英文资料

英文 paper / 海外新闻 / 国际公司:Perplexity / Gemini Deep Research 仍胜。

### ❌ 多模型选择

秘塔走自家路由,**没有 Claude / GPT / Gemini 切换**。
模型对比党 → Perplexity Pro。

### ❌ 极致深度长综述

跑 30-60 分钟出 5000-15000 字综述:Gemini Deep Research 仍是顶级。
秘塔输出**偏中等长度** + 偏快速。

### ❌ Spaces / 工作空间

秘塔没有 Perplexity Spaces / NotebookLM 那种"主题工作空间"。
长期项目 / 主题资料库管理:不如那两个。

## 真实使用 tips

### 1. 不同模式选对场景

秘塔顶部模式切换:
- **简洁**:快速问答
- **深入**:多源 + 详细分析
- **研究**:跑长一点 + 输出综述
- **学术**:学术源优先

实战:
- 一般问题 → 简洁 / 深入
- 重要调研 → 研究模式
- 写论文 / 学术 → 学术模式

### 2. 学术模式 + 引用源

学术模式输出**带 paper 引用**(知网 / Sci-Hub / arXiv 等):
- 你能直接溯源到原 paper
- 写综述时直接用
- 引用格式标准

**学生写论文** 这功能省 1-2 小时/篇。

### 3. 微信小程序很顺手

秘塔小程序:
- 朋友圈刷到信息 → 微信里直接秘塔搜
- 不需切 App

**对碎片信息搜索很顺**。

### 4. 配合 Kimi 探索版

国内 AI 搜索两巨头:
- **秘塔**:搜索专精 + 学术 + 简洁
- **Kimi 探索版**:搜索 + 长文档 + 多轮追问

实战:
- 简短问题 → 秘塔
- 复杂调研 / 上传 PDF 联合 → Kimi 探索版

### 5. 别拿秘塔当主力 LLM

秘塔本质是搜索工具,**不是通用 AI**。
- 写代码 → DeepSeek / Cursor
- 写长文 → Claude / DeepSeek
- 复杂推理 → DeepSeek R1.5 / Claude Opus

### 6. 企业版可对接公司知识库

秘塔企业版:
- 接公司内部知识库
- 私有部署
- 团队协作

适合大企业做"内部知识搜索"。
