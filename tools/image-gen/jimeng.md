## Tier 评级:A

**评级理由**:
- **国内 C 端 AI 图像 default**:用户量国内第一档(和文心一格 / 通义万相竞争)
- **完全免费**(推广期):大额度免费配额,普通用户用不完
- **中文 prompt 母语级**:中国元素 / 中文文字 / 古风 / 二次元都行
- **字节生态联动**:剪映 / 抖音 / 飞书 直接接入
- **多模态闭环**:图 + 视频 + 数字人 一站式

**不是 A+ 的原因**:
- **极致质感** vs Midjourney V7 / FLUX Ultra / Nano Banana **仍有半档差距**
- **商用版权政策**不清晰(用了即梦图做商业项目有 risk)
- **海外用户几乎没认知**
- **数据隐私**(字节体系)对外企用户不友好

## 它**不擅长**什么

### ❌ 极致质感

- 商业广告 / 电商高端品 → FLUX Pro Ultra / Nano Banana 仍胜
- 画廊级艺术 → Midjourney V7 仍胜
- 4K 真实人物写真 → FLUX / Nano Banana 仍胜

### ❌ 风格 / 角色一致性

Midjourney 有 sref / cref。
即梦 4.0 有"参考图"功能但**成熟度不如 Midjourney**。
品牌系列内容 → Midjourney。

### ❌ 商业版权安全

国产 AI 图像产品的**商用版权政策都不清晰**(包括即梦):
- 用户用 AI 生成图做商品 / 广告 / 印刷品
- 法律地位 + 平台政策仍模糊
- 商用场景有 risk

商用项目 → 走 [Adobe Firefly](./firefly.md) 等明确商用 license 的工具。

### ❌ 海外开源 / 协作

即梦海外认知度极低,做开源 / 海外协作时**别人看到"即梦生成"会陌生**。
海外项目 → Midjourney / FLUX。

### ❌ 隐私 / 商业敏感

字节体系数据用于改进模型(协议明确)。
- 公司机密设计
- 商业敏感产品图

慎用即梦(同 [豆包](../foundation-models/doubao.md) 逻辑)。

## 真实使用 tips

### 1. 即梦 vs 文心一格 vs 通义万相 选哪个

国产 AI 图像三家:

| 产品 | 厂商 | 长板 |
|------|------|------|
| **即梦** | 字节 | 综合最强 + 字节生态 + 视频联动 |
| 文心一格 | 百度 | 中国风 / 古风 / 国画 |
| 通义万相 | 阿里 | 阿里生态 + 海外开源(Wan)|

**默认即梦**,**特定中国风场景偶尔切文心一格**。

### 2. 即梦图像 + Hailuo 视频闭环

即梦默认配套字节系的 Hailuo / Seedance 视频:
- 即梦出第一帧 / 静态图
- 一键生成视频

短视频自媒体:**即梦闭环比拼凑工具流畅**。

### 3. 提示词风格

即梦中文 prompt 最稳:
```
中年男性,穿黑色西装,在上海外滩夜景,电影感构图,9:16 竖图
```

可加风格 reference:
- "/写实"
- "/二次元"
- "/古风国画"
- "/扁平插画"

**中文描述 + 风格标签 = 最稳**。

### 4. 配合剪映工作流

剪映里 → AI 工具 → 即梦图生成 + 数字人配音:
- 不用切多个 App
- 不用导出 / 导入
- 可直接接入抖音发布

**对短视频自媒体,这是字节闭环的杀手锏**。

### 5. 商业用途谨慎对待

如果你打算商用:
- 看即梦协议(目前模糊)
- 优先选**商用 license 明确**的:Adobe Firefly / Shutterstock AI / 自部署 FLUX
- 重要项目用即梦做"概念稿",最终交付用其他工具

### 6. 别一直要 4K

即梦 4.0 高分辨率档**消耗免费额度快**。
日常配图(社交媒体 / 文章)**1024x1024 / 1024x1280 完全够**。
真要 4K 商业图 → FLUX Ultra / Nano Banana。
