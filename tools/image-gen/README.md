## 🧭 怎么选

```
你要艺术 / 风格化 / 海外            → Midjourney V7
你要真实 / 文字 / 国内可用 / 自部署 → FLUX Pro Ultra
你已经付 Gemini Advanced            → Nano Banana 白嫖
你要免费 / 中文 / 国内 / 入门        → 即梦
```

**很多专业用户多个都用**:Midjourney 出概念 → FLUX 出商业图 → 即梦做大量批量。
