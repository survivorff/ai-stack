# 🤖 通用 AI Agent / 自动化(AI Agent)

> 非编程专用的 AI Agent / 工作流自动化平台。
> **区别于 [coding-agent/](../coding-agent/)** — 那个是写代码 agent,这个是通用工作流。

---

## 📊 推荐组合(2026-05)

**对话型 Agent / GPTs 类:**
- **[ChatGPT GPTs](../foundation-models/chatgpt.md)**(GPTs 商店)
- **[Claude Projects](../foundation-models/claude.md)**(Claude 项目)
- **[NotebookLM](../research/notebook-lm.md)**(资料专属问答)
- **元宝 / 智谱清言 / 海螺 智能体应用市场**(国产对照)

**视觉化工作流编排(no-code):**
- **n8n**(开源 + 可自部署,2025 年最受欢迎)
- **Make / Zapier**(老牌,价格略高)
- **Coze(扣子)**(字节,国内 + 免费)
- **Dify / FastGPT**(国产开源 + RAG)
- **阿里云百炼 / 火山引擎 / 腾讯云 Agent 工作室**

**Browser Use / 浏览器 Agent:**
- **Comet(Perplexity)**(AI 浏览器)
- **Anthropic Computer Use**(Claude 控制屏幕)
- **OpenAI Operator**(GPT 控制网页)
- **Manus** / **AutoGPT** / **CrewAI**(开源框架)

**国产代表(火热):**
- **Manus**(2025 年现象级国产 Agent)
- **Coze**(字节,工作流 + 智能体一体)
- **Dify**(开源 + 海外影响力)

---

## 一些重要趋势(2026-05)

- **MCP(Model Context Protocol)**:Anthropic 推动的"agent 工具调用"标准,生态最规范
- **Agent / Computer Use**:让 AI"看屏幕 + 点鼠标"。Anthropic 先发,OpenAI / 国产追
- **Agent 编排平台分化**:no-code(Coze / Dify)vs 代码框架(LangChain / LangGraph / CrewAI)
- **国产 vs 海外**:**Coze + Dify 在国内场景已经追到 n8n / Make 同档**
