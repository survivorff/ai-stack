# GLM(智谱)

> 智谱 AI 出的国产开源大模型。**学术血统 + 企业部署成熟**,综合能力均衡。
> Tier A —— 没有特别耀眼的长板,但**没有明显短板,做"企业 / 政务私有化"它常常是首选**。

---

## 基本信息

| 项 | 值 |
|---|---|
| **厂商** | [智谱 AI](https://www.zhipuai.cn)(清华系) |
| **当前版本(2026-05)** | GLM-5(旗舰) · GLM-5 Air(性价比) · GLM-4-9B(开源端侧) |
| **支持模态** | 文本 · 代码 · 图片(理解 + 生成 CogView)· 视频(CogVideoX) |
| **上下文窗口** | 1M tokens(GLM-5) |
| **API 价格** | Input ¥1 / M tokens(GLM-5)· ¥0.5 / M(Air 档) |
| **个人使用** | Web + App 免费 |
| **开源** | ✅ ChatGLM 系列、CogVideoX、CogView 等多个模型 Apache/MIT 协议 |

---

## Tier 评级:A

**评级理由**:
- **国产开源最早的代表**(清华系,从 ChatGLM-6B 时代就在做开源)
- **企业 / 政务的"信任款"**:学术背景 + 长期开源 + 私有化部署成熟
- **多模态自研全栈**:CogView(图)、CogVideoX(视频)、GLM(文本),不依赖第三方
- **Agent / Function Call 的早期推动者**(2023 就在做工具调用)

> **不是 A+ 的原因**:
> - 单项能力**没有"碾压性长板"** — 写代码不如 DeepSeek、长文档不如 Kimi、多模态不如豆包
> - 个人用户层面认知度低
> - 创意写作不出彩
> 它的优势是"**企业级 / 私有部署 / 全栈自研**",**对个人用户体感不强**。

---

## 它最适合做什么

### ✅ 国企 / 政务 / 央国企的私有化部署

智谱在**企业级合规交付**上做得最成熟:
- 模型权重开源 → 完全离线部署
- 公司有完整商业支持团队 → 可签 SLA
- 学术背景 → 在政府 / 国企的"安全审查"通过率高
- 兼容主流 GPU(华为昇腾 / 海光 / 英伟达)

如果你在央国企 / 政府机构 / 军工 / 金融做 AI 项目,**GLM 通常是默认 shortlist**。

### ✅ 学术 / 研究使用

智谱的论文产出和研究透明度**国产里数一数二**:
- 模型卡完整,不藏关键参数
- ChatGLM 系列在 Hugging Face 是中文模型下载量最多的之一
- 学术合作 + 高校研究端用户群大

读 paper / 写综述 / 学术调研:GLM 比某些"营销驱动"的国产模型让人放心。

### ✅ 端侧 / 本地部署

GLM-4-9B / GLM-3-6B **是国产开源里"小而能用"的代表**:
- 笔记本(M2 Mac / 3060+)能跑
- 中文场景质量比 Llama 3 8B 强
- 个人项目 / 学习 / 隐私敏感场景的好选择

LM Studio / Ollama / vLLM 都直接支持。

### ✅ Agent / Function Call

智谱 2023 年就在推 ChatGLM 的 Function Call,生态比较完整:
- 智谱清言里**自带智能体 / Agent 商店**
- API 端 Function Call 稳定性好
- 国产 Agent 框架(LangChain 中文社区)很多默认接 GLM

### ✅ 国产首选的"备份模型"

DeepSeek 出问题 / Kimi 限流时,**GLM 是下一个备用**。
做企业 AI 应用时,**多模型冗余几乎是必须的**,GLM 在国产里稳定性 + 兼容性都是 A 档。

## 它**不擅长**什么

### ❌ "性价比"打不过 DeepSeek

API 价格:
- DeepSeek V4:¥0.5/M(命中缓存 ¥0.05)
- **GLM-5:¥1/M(2 倍)**

**DeepSeek 在"开源 + 性价比"双维度都更狠**。GLM 主要靠**学术信任 + 企业服务**而不是价格。

### ❌ 长上下文打不过 Kimi

GLM-5 1M context 不弱,但**实测在 500K+ 区间答题质量明显下滑**,Kimi 2M 实战更稳。
长文档 → 用 Kimi。

### ❌ 中文创意 / 多模态生成

- 写文案 / 故事 / 营销内容:**Claude / 豆包更出彩**
- 图像生成(CogView):**质量明显落后 Midjourney / FLUX / Nano Banana**
- 视频生成(CogVideoX):**学术意义大于实用,落后 Veo / Sora / Kling**

### ❌ 没有"消费级杀手锏"

DeepSeek 有"价格屠夫" 和 "推理之王" 两个标签,Kimi 有"长文档之王",豆包有"移动端 + 语音"。
**GLM 没有让普通用户记住的标签**,这是它在 C 端持续偏弱的原因。

### ❌ 极复杂代码任务

代码能力中规中矩。**写代码主力 → DeepSeek**(国产)/ Claude(海外)。

## 真实使用 tips

### 1. 智谱清言 vs API,先想清楚定位

智谱有两个产品线:
- **智谱清言**(Web / App):面向 C 端用户,免费,有"智能体"应用市场
- **BigModel API**:面向开发者,企业接入

**别混着用**。
- 个人用 / 学习 / 探索:**智谱清言** + 智能体
- 接入 / 应用 / 开发:**BigModel API**

### 2. CharGLM-3 / CogAgent 是被低估的产品

智谱很多"小模型"在垂直场景表现亮眼:
- **CharGLM-3**:对话 / 角色扮演场景,中文情感细腻
- **CogAgent**:GUI 操作 Agent,能"看屏幕 + 点鼠标"

商业项目里,这些细分模型常常**比通用 GLM 更值钱**。

### 3. 开源版本玩"端侧 / 个人项目"

ChatGLM3-6B / GLM-4-9B 在本地跑非常友好:
- M1 Pro Mac 16GB:跑 9B 量化,**响应速度可用**
- 笔记本 4060/4070:跑 9B 全精度
- 适合做"完全离线"的个人助手 / 学习 / 隐私场景

### 4. 企业用要谈"专属定价"

API 标价是参考价,**企业用量大时,智谱商务谈下来通常能砍 30-50%**。
对比 DeepSeek 几乎是"价格透明无议价空间",GLM 的商业团队反而是优势。

### 5. 别忽略"智能体应用市场"

智谱清言里**有几千个用户做的智能体**(类似 GPTs),**很多在中文细分场景比较好用**:
- 公文写作助手
- 学习辅导专家
- 法律咨询
- 简历优化

**白嫖一下,免费好用就直接用**。

## 价格分析

### 个人(消费者)

| 渠道 | 价格 | 适合谁 |
|------|------|------|
| **智谱清言 Web / App** | **¥0** | 大多数个人用户 |
| **智谱清言 Pro** | ¥99/月起 | 重度 + 优先权 |

**普通用户用免费版完全够**。

### API(开发者 / 企业)

| 模型 | Input | Output |
|------|------:|-------:|
| GLM-5(旗舰) | ¥1 / M | ¥3 / M |
| GLM-5-Air(性价比) | ¥0.5 / M | ¥1.5 / M |
| GLM-4-9B(开源端侧) | 免费(自部署) | 免费 |

### 私有化部署(企业)

智谱企业服务有完整方案:
- 模型 + 算力 + 运维 + 训练数据 + 持续支持
- 价格因规模而异,通常**百万到千万级**
- 在国企 / 政务采购里**几乎是默认选项之一**

### 我的预算建议

```
个人 / 学生:                免费版 + 智能体应用市场
开发者(轻):                 GLM-5 Air ¥0.5/M(配 DeepSeek 当主力)
企业接入(中):                BigModel API,谈专属定价
央国企 / 政务私有化:           商务洽谈 + 自部署,体量大优势大
```

## 一个判断:GLM 适合谁

**适合用 GLM 的画像**:
- 在国企 / 政府 / 金融 / 军工做 AI 集成 → **GLM**
- 学术研究 / 高校项目 → **GLM**
- 想本地部署一个"能用的"中文小模型 → **GLM-4-9B**
- 国产模型多源冗余的"备用第二款" → **GLM**

**不适合 GLM 的画像**:
- 个人开发者纯写代码 → **DeepSeek**
- 个人做长文档 / 调研 → **Kimi**
- 个人做移动端 / 语音 → **豆包**

GLM 不是"个人用户首选",**但在企业 / 学术场景它是国产里很难绕开的选择**。
