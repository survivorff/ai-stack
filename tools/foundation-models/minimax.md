# MiniMax(海螺)

> 稀宇科技 MiniMax 的全家桶。**语音 + 视频生成国产第一档**,海外做了爆款 Talkie。
> Tier A —— 单纯比文本不如 DeepSeek / Kimi,**但多模态(尤其音 + 视频)是国产里最完整的**。

---

## 基本信息

| 项 | 值 |
|---|---|
| **厂商** | [稀宇科技 MiniMax](https://www.minimax.io)(上海) |
| **当前版本(2026-05)** | M2.5(文本) · Speech-02(语音) · Hailuo 02(视频) · Music-02(音乐) |
| **支持模态** | 文本 · 语音(听 + 说 + 克隆) · 视频(生成) · 图(看 + 生成) · 音乐 |
| **上下文窗口** | 1M tokens(M2.5) |
| **API 价格** | Input ¥1 / M(M2.5) · TTS ¥150-300/百万字符 · Video ¥0.5-2/秒 |
| **个人使用** | 海螺 AI / Talkie 免费 |
| **国内可用性** | 🟢 直连(海螺中国版)+ 海外英文版 Talkie / Hailuo |

---

## Tier 评级:A

**评级理由**:
- **语音 / TTS 国产第一**:Speech-02 在中文 TTS 自然度、情感、克隆精度上**国产顶尖**(被 Eleven Labs 海外团队公开认可)
- **视频生成 Hailuo 国产第一档**:Hailuo 02 在角色一致性 + 物理感上**仅次于 Sora 2 / Veo 3**(超过 Kling 2 / Runway)
- **海外影响力**:**Talkie 在 2024-2025 年海外女性用户里现象级**,北美月活一度破 700 万,国产 AI 出海榜样
- **多模态全栈**:文本、语音、视频、音乐、图像,**全自研**(类似阿里 Qwen 全家桶)

> **不是 A+ 的原因**:
> - 文本能力 vs DeepSeek / Kimi 仍有距离
> - 写代码不强
> - 国内 C 端**海螺 App 用户量不如豆包 / 通义**
> - 创意写作"工整有余"
> 它的杀手锏是**音 + 视频**,**纯文本任务不要选它**。

## 它最适合做什么

### ✅ 中文 TTS / 语音克隆

Speech-02 是国产中文 TTS 的天花板:
- **声音自然度**:接近真人,**情感(喜怒哀乐 / 激动 / 平静)细腻**
- **克隆**:30 秒样本能克隆出可用的声音
- **多语言**:中英日韩等
- **延迟低**:实时对话级

适合场景:
- 短视频 / 自媒体配音(替代真人录音)
- 有声书 / 播客 / 音频内容
- 数字人配音
- AI 通话 / 语音助手

**国产做语音 / TTS 项目,MiniMax 几乎是默认 base**。

### ✅ AI 视频生成(Hailuo)

Hailuo 02 在国产消费级视频生成里**第一档**:
- 物理感 + 角色一致性比 Kling 2 / Vidu 略好
- 中文文本理解 + 中国元素(春节 / 茶道 / 古风)比海外模型懂
- 国内可直接用 + 微信支付

适合场景:
- 短视频开头特效片段
- 营销 / 广告概念片
- 个人 AI 视频玩具

**对手 Sora 2(有内容审核 + 海外限制) / Veo 3(海外为主)**,Hailuo 在国内场景胜出。

### ✅ 情感陪伴 / AI 角色

MiniMax 的 **Talkie / 星野**(中文版)是国产 "AI 角色陪伴" 头部产品:
- 角色塑造能力强(角色性格、记忆、互动)
- 情感细腻
- 海外 Talkie 在北美女性用户群里现象级

**做 AI 角色 / 陪伴 / 情感产品**,MiniMax 是国产首选。

### ✅ 短视频内容创作

**MiniMax = 视频脚本(M2.5)+ 配音(Speech-02)+ 视频画面(Hailuo)+ 背景音乐(Music-02)** 全闭环。
做短视频自媒体 / 抖音号,**不需要拼凑多家工具**。

### ✅ 海外项目 / 出海团队

MiniMax 是**国产 AI 出海最成功的之一**(Talkie 在北美女性用户里曾经现象级)。
做出海产品 / 海外用户:**MiniMax 海外版本 + 国内 API,体验和合规都成熟**。

## 它**不擅长**什么

### ❌ 复杂代码 / 推理

文本主力是 M2.5,**vs DeepSeek V4 / Claude / GPT 仍有差距**。
- 写代码 → DeepSeek / Claude
- 推理 → DeepSeek R1.5 / Claude / GPT-5

### ❌ 严肃长文写作

写公众号长文 / 论文 / 严肃专栏:**Claude / DeepSeek 更稳**。
MiniMax 适合"短 / 趣 / 多媒体" 内容,不适合"长 / 严肃 / 深度" 文字。

### ❌ 长文档调研

长上下文虽 1M,但实战体验 vs Kimi 2M / Gemini 1M 仍有差距。
长文档 → Kimi / Gemini。

### ❌ "性价比绝对值"

API ¥1/M 文本 vs DeepSeek ¥0.5/M。
TTS / Video API 单价**远高于纯文本**(因为模态贵)。
极致省钱 → DeepSeek 文本 + 自部署模型;
但**音 / 视频 / 多模态闭环**,MiniMax 性价比仍合理。

### ❌ 个人 C 端的"日常对话主力"

海螺 App 体验**不如豆包流畅**,日常 chat 不如豆包 / Kimi 顺手。
**MiniMax 不是"日常对话主力",是"创作工具 / 情感陪伴"**。

## 真实使用 tips

### 1. Speech-02 试用直接走 MiniMax 官网

[MiniMax 开放平台](https://platform.minimax.io)右上角直接体验 TTS。
- 上传 30 秒 wav → 立即生成克隆音色
- 多种音色试听
- 调情感 / 语速 / 停顿
- 导出 mp3

**做自媒体配音的别再花 100 元请录音了**,Speech-02 几块钱搞定。

### 2. Hailuo 视频先用文字描述清楚"运镜"

很多人 prompt:"一只猫在草地上"。结果片不动 / 不像猫。
**好的 prompt 要写运镜**:

```
镜头:从俯拍缓慢推近到平视
主体:一只橘猫,绿色虎纹眼睛,在阳光下伸懒腰
环境:夏日草地,微风,远处有蒲公英飞
风格:电影感,暖色调,DOF 浅景深
时长:5 秒,前 2 秒猫睁眼,后 3 秒抓蒲公英
```

**Hailuo 的物理感会让你感觉是"导演实拍"**,不是 AI 飘。

### 3. 短视频自媒体的"三件套工作流"

```
脚本(MiniMax M2.5 / 豆包 / DeepSeek)
   ↓
分镜文字(MiniMax M2.5)
   ↓
画面(Hailuo / 剪映 AI / 即梦)
   ↓
配音(Speech-02 克隆你自己声音)
   ↓
背景音乐(Music-02)
   ↓
剪辑组合(剪映)
```

**MiniMax 占了其中 4 步**,单工具能跑完短视频生产链。

### 4. Talkie / 星野 适合做产品分析

如果你做 AI 产品 / AI 陪伴,**别只盯 ChatGPT / Character.ai**。
**去 Talkie 看一周**,你会理解为什么"角色 + 故事感 + 情感细腻"在情感陪伴是核心,而不是"模型聪明"。

### 5. API 走"火山引擎 / 阿里云"反而更便宜

火山引擎 / 阿里云百炼**也接入了 MiniMax 模型**。
量大时,**通过云厂商二次议价比直接走 MiniMax 官方渠道便宜 20-40%**。

## MiniMax 适合谁

**核心定位**:**多模态创作 + 情感陪伴产品**。

**最适合的人**:
- 做短视频 / 自媒体内容(脚本 + 配音 + 视频一站式)
- 做 AI 角色 / 陪伴 / 故事产品
- 出海团队(海外有用户基础)
- 需要中文 TTS / 语音克隆的所有项目

**不适合**:
- 写代码主力(用 DeepSeek)
- 长文档调研(用 Kimi)
- 严肃长文写作(用 Claude)

MiniMax 是国产 AI 里**做得"最像艺术家而非工程师"的一家**,这是它的特色,也是它的边界。
