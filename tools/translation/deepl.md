# DeepL

> 老牌专业翻译工具(2017 起步)。**翻译质量"自然度"长期超过 Google Translate**。
> Tier A —— **2024-2026 被 LLM(Claude / GPT)在某些场景反超**,但依然是"专业翻译"的稳定选择。

---

## 基本信息

| 项 | 值 |
|---|---|
| **厂商** | [DeepL SE](https://www.deepl.com)(德国) |
| **形态** | Web / 桌面 / 移动 / 浏览器扩展 / API |
| **支持语言** | 30+ 语言互译 |
| **国内可用性** | 🟢 网络通,Stripe 卡可付 |

---

## Tier 评级:A

**评级理由**:
- **专业翻译稳**:多年积累,**法律 / 学术 / 商业文档翻译质量稳定**
- **语言覆盖广**:30+ 语言互译,小语种支持比 LLM 派全
- **API + 文档翻译**:Pro 用户可上传整个 PDF / Word 文档 → 保留格式翻译
- **DeepL Write**:语法 / 改写工具(对学术英文写作有用)

**不是 A+ 的原因**:
- **中文 vs Claude / GPT 已被反超**(尤其口语化 / 创意翻译)
- **价格**(¥39+ 月费)** vs 沉浸式翻译 + Claude API 不划算**
- **极口语 / 网络梗 / 文化语境** 不如 LLM
- **2026 年很多专业用户已经迁到 Claude / GPT 翻译**

## 它最适合做什么

### ✅ 法律 / 商业 / 学术文档翻译

- 合同 / 法律文件
- 学术论文翻译
- 商业报告
- 政府文件
- 医学 / 技术专业翻译

DeepL 在这些场景**翻译稳 + 术语准 + 格式保留好**。
LLM 派这块**略次**(术语偶尔自由发挥)。

### ✅ 文档(PDF / Word / PPT)翻译

DeepL Pro 可上传整文档:
- 保留原格式
- 一键翻译整篇
- 导出译文

**对手 LLM 类工作流要"复制粘贴"**,DeepL 这条更顺。

适合:
- [商务翻译](../../scenarios/business-translation.md)
- [快速翻译](../../scenarios/quick-translation.md)

### ✅ 30+ 语言互译

- 阿拉伯语 / 印尼语 / 泰语 / 越南语 / 土耳其语等
- 中欧 / 北欧 / 东欧小语种

**LLM(Claude / GPT)主要中英 + 几大语言**。
真的小语种翻译 → DeepL 仍胜。

### ✅ 老用户工作流

如果你已经在用 DeepL 多年:
- 工作流嵌入(浏览器扩展 / 桌面快捷键)
- 不想换习惯
- → **继续用,没必要折腾切**

## 它**不擅长**什么

### ❌ 中文 vs LLM 派的对比

2026 年很多专业用户**已经切到 Claude / GPT 翻译中文**:
- Claude 中文最自然
- GPT 中文综合稳
- DeepL 中文偶尔生硬(机翻味)

中文为主 → 直接 Claude / GPT 网页 prompt:
> 帮我翻译这段中英文对话,要自然、像真人写的

### ❌ 创意 / 营销 / 文学翻译

DeepL 偏"机器准确",**创意翻译**(广告 / 营销 / 文学 / 网络梗)→ Claude / GPT 更好。

### ❌ 极致省钱

LLM 翻译月费(走 DeepSeek 或 Claude API):**几乎免费**。
DeepL 月费 ¥39+ 仍贵。

### ❌ 上下文 / 长文翻译保持一致

LLM 做长文翻译时**能保持上下文 / 术语一致**。
DeepL 是**逐句翻**,长篇略碎。

## 真实使用 tips

### 1. DeepL vs Claude / GPT 翻译怎么选

简化决策:
- **法律 / 学术 / 严肃专业** → DeepL
- **博客 / 营销 / 创意 / 中英** → Claude / GPT
- **小语种(非英中日韩) → DeepL
- **快速看一段网页 / 邮件** → 沉浸式翻译(双引擎)

### 2. 沉浸式翻译可以同时调 DeepL + Claude

[沉浸式翻译](./immersive-translate.md)是浏览器扩展:
- 可同时调 DeepL + Claude / GPT 多个引擎
- 双引擎对照
- 选最合适的

**对挑剔翻译质量的用户:这是最佳工作流**。

### 3. DeepL Write 学英语 / 改稿

[DeepL Write](https://www.deepl.com/write):
- 输入英文 → AI 改成更地道
- 类似 Grammarly + 风格优化

**对中国人写英文邮件 / 论文 / 商务文档** 友好。

### 4. 桌面 App 快捷键

DeepL 桌面 App:
- Cmd+C+C(双击)= 翻译选中
- 全局快捷键
- 不打断工作流

**比浏览器扩展更顺**(能在任何 App 里用)。

### 5. API 接产品

DeepL API:
- 每月 50 万字符免费
- 适合接产品做"嵌入式翻译"
- 对手 Google Translate API / OpenAI 翻译

跨境电商 / 多语言产品:**DeepL API 是默认选择之一**。

### 6. 长文档翻译 Pro 起

DeepL 文档翻译要 Pro($30/mo)。
偶尔翻译 → Claude / GPT 网页 + 复制粘贴足够省钱。
