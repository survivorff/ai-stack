# ✍️ 写作工具(Writing)

> 长文写作 / 改稿 / 创意写作。

---

## 这个分类是什么、不是什么

✅ **是**:专门面向"长文写作"的工具
- Notion AI(已在 [notes/](../notes/notion-ai.md))
- 飞书智能伙伴(已在 [notes/](../notes/feishu-ai.md))
- 火山写作(字节,见下)
- 秘塔写作猫
- Grammarly / DeepL Write(改稿)

❌ **不是**:模型本身(那是 [foundation-models/](../foundation-models/) — Claude / GPT 写作能力最强)

---

## 📊 推荐组合(2026-05)

**写中文长文 / 公众号 / 知乎:**
- 主力:**[Claude Sonnet 4.7](../foundation-models/claude.md)**(中文最自然) + **[DeepSeek](../foundation-models/deepseek.md)**(国内 + 备用)
- 辅助:**[飞书智能伙伴](../notes/feishu-ai.md)**(团队)/ Notion AI(个人)
- 改稿:火山写作 / 秘塔写作猫

**写英文长文 / blog / academic:**
- 主力:**[Claude Sonnet 4.7](../foundation-models/claude.md)** / **[GPT-5](../foundation-models/chatgpt.md)**
- 改稿:**Grammarly** / **[DeepL Write](../translation/deepl.md)**

**严肃写作工作流详细见 [scenarios/long-form-writing.md](../../scenarios/long-form-writing.md)**(待写)。

## 待详写

| 工具 | 厂商 | 长板 | 优先级 |
|------|------|------|--------|
| 火山写作 | 字节 | 国内 + 免费 + 公众号场景 | 中 |
| 秘塔写作猫 | 秘塔 | 中文改稿 / 病句 / 重复 | 中 |
| Grammarly | Grammarly | 英文改稿 + 浏览器扩展 | 中 |
| Sudowrite | 个人开发 | 小说 / 创意写作 | 低 |
| Jasper / Copy.ai | Jasper | 营销文案(已被 Claude / GPT 替代) | 低 |

> 写作的本质工具是**模型本身(Claude / GPT / DeepSeek)**。
> "写作专属产品"很多是模型的"产品壳",直接用模型 + 好的 prompt 反而更灵活。
> 因此本分类暂不做大量深拆,优先级排在 [foundation-models](../foundation-models/) 之后。
