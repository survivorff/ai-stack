## Tier 评级:A+

**评级理由**:
- **国产视频生成第一档**:Kling 3 综合质量 vs Sora 2 / Veo 3 仍略弱(差 1-2 档),**但已经远超 2024 年的所有产品**
- **国内零门槛**:不翻墙、微信支付、中文母语 prompt
- **中国元素 / 古风 / 二次元**:风格覆盖国内场景最完整
- **完整工作流**:首尾帧 + 多镜头 + 视频编辑 + 数字人,生态完整
- **快手生态**:可直接发布到快手 / 抖音

**不是 S 的原因**:
- **物理感 vs Sora 2 仍有差距**(尤其水流 / 布料 / 复杂动作)
- **海外用户认知度**远低于 Sora / Veo
- **商用 license 政策模糊**(同其他国产)
- **高分辨率 / 长视频** 需会员

## 它**不擅长**什么

### ❌ Sora 2 / Veo 3 第一档质量

旗舰 PK 旗舰:Sora 2 / Veo 3 仍胜。
专业级广告 / 商业大片:海外旗舰更稳。
不过 90% 内容创作场景 Kling 3 已经够。

### ❌ 长视频(30 秒+)

Sora 2 60 秒 vs Kling 3 主要 5-10 秒(Pro 30 秒)。

### ❌ 海外协作

海外用户 / 海外平台对 Kling 认知度低。
做海外项目 → Sora / Veo 仍是主流。

### ❌ 隐私 / 商业敏感

数据训练协议(同字节系)。
商业敏感视频 → 慎用。

## 真实使用 tips

### 1. Kling vs Hailuo vs 即梦视频

国产视频三巨头:

| 产品 | 厂商 | 长板 |
|------|------|------|
| **Kling** | 快手 | 综合质量第一 + 工作流完整 |
| **Hailuo** | MiniMax | 物理感 + Talkie 海外影响 |
| **即梦视频** | 字节 | 剪映 / 抖音生态闭环 |

**默认 Kling**,**特定字节生态用即梦,需要 MiniMax 闭环用 Hailuo**。

### 2. 首尾帧用法

最实用的工作流:
- [Nano Banana / FLUX](../image-gen/) 出"开始帧"和"结束帧"
- Kling 首尾帧 → 中间动画补全
- **过渡稳定 + 动作可控**

### 3. 风格控制

Kling 3 自带风格 preset:
- /电影感
- /二次元
- /国漫
- /Vlog
- /广告
- /MV

**比纯 prompt 描述风格稳得多**。

### 4. 提示词节奏

Kling 3 喜欢:
```
镜头:从近景到远景,缓慢推开
主体:女孩在樱花树下闭眼微笑
环境:春日下午,阳光斜照,微风,花瓣飞舞
风格:日系治愈,浅景深,温暖色调
时长:5 秒
```

**显式写运镜 + 时序**。

### 5. 商业项目走"专属版权" 服务

Kling 提供:
- 普通会员(个人创作)
- **企业版**(商用 license + audit)

商用项目 → 走企业版谈定价。

### 6. 配合 Suno / 海螺 加音

Kling 输出**没有原生音效**(vs Veo 3)。
工作流:
- Kling 出无声视频
- [Suno](../audio/) 加 BGM 或 [海螺 / Eleven Labs](../audio/) 加配音
- 剪映合成
