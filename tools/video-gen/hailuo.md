# Hailuo(海螺)

> MiniMax 出的 AI 视频生成。**物理感国产第一**(超过 Kling),海外有 hailuoai.com 双用户基础。
> Tier A+。

## Tier 评级:A+

**评级理由**:
- **物理感国产第一**:Hailuo 02 在水流 / 布料 / 人体动作的真实度上**国产顶尖**(略胜 Kling)
- **海外英文版同步**:hailuoai.com 有海外用户基础(虽然主要是 Talkie 带过来)
- **MiniMax 多模态闭环**:配套 Speech-02(TTS)+ Music-02(BGM)+ M2.5(脚本)= 短视频一站式
- **出海项目友好**:海外身份 / 海外卡可用,中国大陆也直连
- **5 秒短视频质量天花板**

**不是 S 的原因**:
- **综合 vs Sora 2 / Veo 3** 仍略弱半档
- **长视频(15+ 秒)** 不如 Kling
- **风格丰富度**(二次元 / 国漫 / 古风)不如 Kling
- **C 端 App 用户量** 国内不如 Kling 多

## 它最适合做什么

### ✅ 物理感强 / 真实场景

- 自然现象(水 / 风 / 火 / 烟)
- 人物动作(走 / 跑 / 抱 / 转身)
- 物体物理(掉落 / 撞击 / 翻转)

国产里要"最像真拍的" → Hailuo 02。

### ✅ MiniMax 多模态闭环

工作流:
- M2.5 写脚本
- Hailuo 02 出视频
- Speech-02 配音(可克隆你声音)
- Music-02 加 BGM
- 一站式

**对手 Kling 没有这种"多模态原生闭环"**,这是 MiniMax 的差异化。

### ✅ 出海团队 / 海外协作

MiniMax 是国产 AI 出海最成功的(Talkie 现象级)。
做出海产品的国内团队:Hailuo 跨境合规 + 用户认知度都比 Kling 好。

### ✅ 5-6 秒短片质量

5-6 秒视频:**Hailuo 02 单帧质量 + 物理感**国产最强。
做"短特效片段 / 视频开头 / 营销 5 秒大片":Hailuo 02 是首选。

## 它**不擅长**什么

### ❌ 长视频(15+ 秒)

Sora 2 60 秒 / Kling 3 30 秒 / Veo 3 15 秒 / **Hailuo 主要 6-10 秒**。
长视频 → Sora / Kling。

### ❌ 风格 / 二次元 / 古风

Hailuo 偏"真实物理",**风格化弱于 Kling**:
- 二次元 / 国漫 → Kling / 即梦动漫
- 古风 / 中国元素 → Kling / 文心一格 / 即梦
- 极电影感 → Sora / Veo

### ❌ 国内 C 端用户量

国内 C 端用户量:Kling > 即梦视频 > Hailuo。
团队协作时同事可能更熟 Kling。

### ❌ 隐私 / 商业敏感

同其他国产,数据隐私需谨慎。

## 真实使用 tips

### 1. 主要用 6 秒,质量最稳

Hailuo 02 在 6 秒视频上质量最稳。
要 10+ 秒 → 拼接多段 6 秒(MiniMax 内置剪辑)。

### 2. 配 Speech-02 做"AI 数字人"

Hailuo + Speech-02:
- 出视频(无声)
- Speech-02 用你的声音配旁白
- 合成 = AI 主播

**比拼凑工具(D-ID + Eleven Labs)便宜 + 顺**。

### 3. 国内 vs 海外版

- **国内版**(MiniMax 海螺):中文 prompt + 微信支付
- **海外版**(hailuoai.com):英文 prompt + Stripe 卡

国内项目走国内版,出海项目走海外版,**两个账号互通**(MiniMax 同账号体系)。

### 4. 提示词节奏

Hailuo 喜欢"具体物理动作":
```
0-2s:女子手拿茶杯,蒸汽缓缓升起
2-4s:她吹一下杯口,蒸汽飘散
4-6s:她抿了一口,微笑

镜头:中景,平视,固定
环境:傍晚咖啡馆,暖光,远处人模糊
```

**比"美丽女子喝茶"高质 10 倍**。

### 5. 配合 Music-02 + Speech-02 做完整短片

```
M2.5(脚本)
  → Hailuo 02(视频)
  → Speech-02(配音克隆)
  → Music-02(BGM)
  → 剪映 / Final Cut(合成)
```

**单平台 4 个模型,自媒体生产线**。

### 6. 别拿 Hailuo 做长剧情

Hailuo 适合"片段 / 特效 / 镜头" — **不擅长讲故事**。
要做长剧情(30 秒+)→ 用 Sora 2 / Kling 3,或多段 Hailuo 拼接 + 后期编排。
