# Suno

> 文生歌(Text-to-Song)的天花板。**给一段歌词或 prompt → 出一首带词、带旋律、带编曲的完整歌**。
> Tier S。**V5 已经做到"非专业耳朵听不出是 AI"**。

## Tier 评级:S

**评级理由**:
- **文生歌行业第一**:V5 输出**非专业耳朵几乎听不出 AI**
- **一站式**:歌词 + 旋律 + 编曲 + 演唱,**单工具完整生产**
- **风格广**:流行 / 摇滚 / 古典 / 嘻哈 / 民谣 / 国风 / 二次元都行
- **入门门槛 0**:输入 prompt → 30 秒出歌

> 不是无短板:
> - 中文吐字仍偶尔不清(中文歌质量略弱英文)
> - 商用 license 要 Pro 起
> - 法律 / 版权问题仍在演进
> - 国内访问门槛

但**作为消费级 AI 音乐生成,Suno 是事实标准**。

### ✅ 营销 / 广告 jingle

- 品牌歌
- 活动主题曲
- 30 秒广告片头 / 片尾

成本 ¥30/月 vs 雇音乐人 ¥几千起步。

### ✅ 个人创作 / 玩玩

- 给朋友 / 家人写生日歌
- 表白 / 求婚特别曲
- 公司年会 / 团建曲
- 单纯好玩

入门门槛极低 + 出片快。

V5 的英文歌质量好 → 拿来当听力素材 / 跟唱练习。

## 它**不擅长**什么

### ❌ 专业级商业音乐

- 唱片 / 商业演出级 → 仍要专业制作
- 极致 mixing / mastering → 后期专业人士
- 真人歌手"个性化嗓音"→ 无法复刻

### ❌ 中文歌完美吐字

V5 中文歌吐字仍偶尔模糊 / 重音错位。
要中文歌发布:**生成多次挑最好的**,或后期补音轨。

### ❌ 完全自定义旋律

Suno 给"AI 写歌",**你不能精确指定旋律**(只能描述风格)。
要严格控制 → 用专业 DAW(Logic / Ableton)+ 真人创作。

### ❌ 国内零门槛

需要海外 IP + Stripe 卡。

### ❌ 低预算长期商用

商用要 Pro $30/月。少量商用 → 划算;大批量长期 → 仍可能买正版库便宜。

## 真实使用 tips

### 1. Custom 模式 vs Simple 模式

**Simple**:输入 prompt(主题 + 风格),AI 全包。
**Custom**:你自己写歌词 + 描述风格 + 控制结构。

```
Simple:"一首伤感的中文民谣,关于离别的故乡"
→ AI 全自动写词 + 编曲

Custom:
[Verse 1]
我离开了那条河
她还在山间流过
...

[Chorus]
带上一颗故乡的心
...

Style: 中文民谣,木吉他 + 钢琴,90 BPM,伤感
→ AI 用你的词 + 你描述的风格
```

**严肃创作用 Custom**,**玩玩用 Simple**。

### 2. 风格描述要具体

❌ "好听的歌"
✅ "Indie folk, warm acoustic guitar, female vocal, 80 BPM, intimate, cafe vibe"

**具体的风格 + 节奏 + 情绪 + 乐器** = 高质量输出。

### 3. 多生成几次挑

每次 Suno 生成 2 条候选。**生成 3-5 次**,从 6-10 条里选。
**不要第一条就用** —— 有时候第二次更好。

### 4. 用 Extend 功能做长歌

Suno 单次输出 2-3 分钟。要更长(完整 3-4 分钟):
- 生成第一段
- 用 Extend 续写
- 接到自然结束

### 5. 配合视频工具的工作流

```
脚本(GPT / Claude)
  → 视频(Sora / Veo / Kling)
  → BGM(Suno)
  → 配音(Eleven Labs / Speech-02)
  → 剪辑(剪映 / Final Cut)
```

**Suno 在自媒体生产线里是音乐这环的 default**。

### 6. 国内合理姿势

- 海外身份 + Stripe → 直接付 $10
- 不愿折腾 → [MiniMax Music-02](./music-02.md)(国产替代,免费档可用)
