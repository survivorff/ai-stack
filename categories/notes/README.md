## 我的判断

### 不同场景的最优解

| 场景 | 用什么 |
|------|--------|
| 一份长资料快速消化 | NotebookLM(免费且最准) |
| 日常 dump + 自动归类 | Mem |
| 系统性知识管理 | Tana(陡峭学习曲线但威力大) |
| 工作场景(团队) | 飞书智能伙伴(国内首选) |
| 写作型每日笔记 | Reflect |
| 本地隐私 + 大量插件 | Obsidian |

### 我自己用什么

```
工作:飞书(团队)
日常:Tana(主仓库)
临时 dump:Apple Notes(轻量)
研究消化:NotebookLM(单次任务)
```
