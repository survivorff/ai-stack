## 🔄 2026 年这一年的赛道变化

### 关键事件

| 时间 | 事件 |
|------|------|
| 2025-12 | Cursor 3.0,推出 Composer Mode,多文件并行编辑 |
| 2026-02 | Anthropic Claude Sonnet 4.7 发布,Claude Code 同步升级 |
| 2026-03 | Windsurf 涨价(原 $15→$20),向 Cursor 看齐 |
| 2026-04 | Devin 公开,自主开发 agent,$500/mo,大厂在试 |
| 2026-05 | Cursor 推出 Ultra ($200/mo),配合 Background Agents |

### 我的判断

- **2 强格局已成**:Cursor(IDE)+ Claude Code(CLI),其他都是这两个的"特定场景替代"
- **国内化趋势**:豆包、通义都在做编程模型,国内开发者会越来越少需要 Cursor
- **Agent 化**:Devin 不是"工具",是"团队成员",2026 下半年起会有 SaaS 化趋势
- **价格上涨已成定局**:重度用户月费会到 $50-200
