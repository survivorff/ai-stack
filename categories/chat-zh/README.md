## 我的判断

### 国内开发者的"够用"组合

```
日常问答:豆包 (App 体验)
长文档:Kimi (200K context)
代码:DeepSeek 或通义灵码
工作场景:看你公司用什么(飞书 / 钉钉 / 企微)
```

**总成本:¥0**。这是国内开发者的最大优势。

### 什么时候还需要海外大模型

- 写英文内容 → Claude
- 复杂代码任务 → Claude Code(国内开发者强烈推荐自配代理)
- 多模态高级场景 → Gemini / GPT
