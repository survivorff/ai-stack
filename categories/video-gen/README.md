## 我的判断

### 不同场景的最优解

| 场景 | 用什么 |
|------|--------|
| YouTube / 营销视频 | Veo 3.1 |
| 短叙事(故事性) | Sora 2 |
| 国内合规需求 | Kling 3.0 |
| 镜头精确控制(广告) | Runway Gen-4.5 |
| 数字人讲解 | HeyGen |
| 快速 < 10 秒社交内容 | Pika 2.0 |

### 这一年的变化

- **2024-12 Sora** 公开,引爆赛道
- **2025-2026** 谷歌、字节、腾讯、Anthropic 全部入场
- **当前**:Veo 3.1 在多数 benchmark 第一,Sora 2 在故事感第一
- **国内**:Kling、Hailuo、Seedance 已经追平 70% 海外能力
