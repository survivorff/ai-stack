## 我的判断

### 不同场景的最优解

| 场景 | 用什么 |
|------|--------|
| 博客封面 / 营销图 | Midjourney + Photoshop 微调 |
| 真实产品图 | FLUX Pro |
| 带文字的海报 / Banner | Ideogram 3 |
| 商业用 / 客户交付 | Adobe Firefly(版权清晰) |
| 矢量 / Logo | Recraft |
| 想免费试试 | Ideogram 3 免费版(每天 10 张) |

### 这一年的变化

- **2025-12 Nano Banana**(Google Gemini 3 Pro Image)出来,首次让"4K 真实感"成为可能
- **2026-01 GPT Image 2** 发布,文字渲染问题终于解决
- **2026-03 FLUX Pro 1.5** 真实感超越 Midjourney(在写实场景)
- **Midjourney V8** 仍在路上,2026 上半年依然是艺术风格 No.1
